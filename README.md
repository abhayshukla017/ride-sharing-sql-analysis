#  Ride-Sharing SQL Data Analysis Project

##  Project Overview
This project analyzes ride-sharing trip data to understand revenue trends,
city performance, driver efficiency, and customer behavior using SQL.

##  Dataset
- Rows: 1000
- Primary Key: ride_id (unique)
- Source: Synthetic dataset (created for learning)

##  Database Design
- ride_sharing_data (raw table)
- drivers
- customers
- rides

##  Tools Used
- MySQL
- MySQL Workbench
- Git
- GitHub

##  Analysis Performed
- Revenue and city performance analysis
- Driver productivity and cancellation rate
- Customer lifetime value (CLV)
- Payment method trends
- Advanced SQL (CTEs, Window Functions)

##  Key Skills
- Joins
- Aggregations
- Window Functions
- CTEs
- Business Insights
